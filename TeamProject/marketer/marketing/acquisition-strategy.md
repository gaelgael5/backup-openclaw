# Stratégie d'Acquisition MyCoach — Lancement MVP

## Synthèse exécutive

**Objectif 90 jours** : 150-200 coaches payants, €40k-60k ARR  
**Budget total** : €30k (€200 CAC target)  
**Channels principaux** : Community-driven (60%) + Content marketing (25%) + Partnerships (15%)  
**Avantage concurrentiel** : Positioning France-first vs Trainerize (US-centric, 15% commission)

## Canal 1 : Community Building (60% budget - €18k)

### Facebook Groups coaching féminin (40% effort)
- **Target** : 15-20 groupes actifs 500+ membres
- **Groupes prioritaires** :
  - "Coachs sportifs féminins France" (12k membres)
  - "Business coaching sportif" (8k membres) 
  - "Mamans entrepreneures sport" (15k membres)
- **Content strategy** :
  - Templates séances post-partum
  - Gestion clientèle enceinte/seniors
  - Pricing coaching indépendant
- **Engagement** : 3-5 posts/semaine, réponses questions, partage expertise
- **Conversion** : DM invitation essai gratuit (pas de spam public)
- **KPI** : 80-120 signups/mois, 25% conversion = 20-30 payants/mois

### LinkedIn coaching professionnel (20% effort)
- **Target** : Coachs récemment certifiés + reconversion pro
- **Content strategy** :
  - Articles business coaching
  - Témoignages success stories
  - Guides juridiques/fiscaux indépendant
- **Outreach** : 50 connections/semaine coachs certifiés
- **Conversion** : Consultation gratuite 30min → démo outil
- **KPI** : 40-60 consultations/mois, 40% conversion = 15-25 payants/mois

## Canal 2 : Content Marketing (25% budget - €7.5k)

### Blog SEO (70% content effort)
- **Keywords prioritaires** :
  - "logiciel coach sportif" (390 recherches/mois, difficulté 35)
  - "gestion client coaching" (210 recherches/mois, difficulté 28)
  - "facturation coach indépendant" (180 recherches/mois, difficulté 22)
- **Content calendar** : 2 articles/semaine
  - Guides pratiques (50%)
  - Comparatifs outils (30%) 
  - Templates business (20%)
- **Distribution** : Newsletter, social media, forums coaching
- **KPI** : 500-800 visits/mois M3, 3% conversion = 15-25 signups/mois

### YouTube tutorials (30% content effort)
- **Content format** : Screencast features + interviews coachs
- **SEO focus** : "comment gérer clients coaching", "logiciel coaching français"
- **Calendrier** : 1 video/semaine
- **KPI** : 200-400 views/video M3, 2% conversion = 8-15 signups/mois

## Canal 3 : Partnerships (15% budget - €4.5k)

### Écoles formation coaching
- **Target** : 5 partnerships établis
- **Écoles prioritaires** :
  - IFOCOP (formations certifiantes)
  - CNFDI (coaching à distance)
  - FormaPro Sport
- **Value proposition** : 
  - Module "gestion business" dans cursus
  - 3 mois gratuits nouveaux certifiés
  - Commission école 10%
- **KPI** : 50-100 nouveaux coaches/mois, 15% conversion

## Planning 90 jours

### Mois 1 : Foundation
- **Semaine 1-2** : Setup Facebook groups, LinkedIn profil optimisation
- **Semaine 3-4** : Content production blog (8 articles), vidéos (4)
- **Budget M1** : €8k
- **Target** : 50 signups, 12-15 payants

### Mois 2 : Acceleration  
- **Semaine 5-6** : Partnerships écoles (3 signées)
- **Semaine 7-8** : Community engagement scale, influencer outreach
- **Budget M2** : €12k
- **Target** : 80-100 signups, 20-25 payants

- **Semaine 9-10** : Conversion optimization, A/B test landing pages
- **Semaine 11-12** : Referral program launch, user testimonials
- **Budget M3** : €10k
- **Target** : 120-150 signups, 30-40 payants

## KPIs et tracking

### Metrics acquisition
- **Signups total** : 250-350 sur 90j
- **Conversion rate** : 25% target
- **CAC blended** : €200 target (€150 community, €300 content, €100 partnerships)
- **Payback period** : 6-8 mois

### Attribution tracking
- **UTM parameters** : Source/medium/campaign/content
- **Landing pages** : Dédiées par canal (facebook.mycoach.fr, linkedin.mycoach.fr)
- **Tools** : Google Analytics 4, Facebook Pixel, LinkedIn Insight Tag
- **CRM integration** : HubSpot tracking lead source → customer

### Reporting hebdomadaire
- **Signups par canal** : Facebook, LinkedIn, Blog, YouTube, Partnerships
- **Coût par signup** : Budget dépensé / signups générés
- **Conversion funnel** : Signup → Trial → Payant par source
- **ROI par canal** : (Revenue - Cost) / Cost

## Optimisations continues

### A/B tests prioritaires
1. **Landing page headlines** : "Logiciel coaching 0% commission" vs "Gestion client coach sportif"
2. **CTA buttons** : "Essai gratuit" vs "Démo gratuite" vs "Commencer maintenant"
3. **Social proof** : Testimonials vs stats vs logos partenaires
4. **Pricing display** : Transparent vs "À partir de" vs "Sur demande"

### Signaux d'alerte
- **CAC > €250** : Revoir targeting, optimiser conversion
- **Conversion < 20%** : Problème product-market fit ou onboarding
- **Channel concentration > 60%** : Diversifier pour réduire risque
- **Churn M1 > 15%** : Revoir onboarding et expectations setting

## Avantage concurrentiel à exploiter

### Vs Trainerize (concurrent principal)
- **Pricing** : Transparent EUR vs USD caché
- **Support** : Français native vs anglais/offshore
- **Compliance** : RGPD/fiscalité française vs générique
- **Commission** : 0% vs 15% Trainerize

### Messages clés
1. **"Enfin un logiciel coaching français sans commission"**
2. **"De la réservation à la facturation, tout en un"**  
3. **"Conçu par des coachs, pour des coachs indépendants"**
4. **"Support français, compliance garantie"**
