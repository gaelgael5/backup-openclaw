# MyCoach UX Designs

Designs UX ultra-modernes pour MyCoach avec glassmorphism et effets néon.

## Design System

### Couleurs

**Coach (Interface Dark/Tech) :**
- Primary: #0A0E1A (noir profond)
- Accent: #7B2FFF (violet électrique) 
- Secondary: #FF6B2F (orange énergie)

**Client (Interface Light/Dynamique) :**
- Primary: #F0F4FF (blanc bleuté)
- Accent: #00C2FF (cyan dynamique)
- Secondary: #FF6B2F (orange énergie)

### Effets

- Glassmorphism 4 niveaux
- Effets néon différenciés
- Micro-animations 200+ concepts
- Transitions fluides 60fps

## Écrans Principaux

- Onboarding immersif
- Dashboard Coach (cyber command center)
- Dashboard Client (lifestyle gaming)
- Navigation différenciée

## Inspiration

- Apple Vision Pro glassmorphism
- Tesla Model S interface cyber
- Nike Training Club motivation
- Strava achievements social
