# MyCoach CI/CD Pipeline

GitHub Actions workflows pour backend et frontend.

## Backend CI/CD

### Pipeline FastAPI
- Tests PostgreSQL + Linting
- Sécurité Trivy scan
- Build Docker image
- Deploy staging/production
- Notifications Discord

## Frontend CI/CD  

### Pipeline Flutter
- Tests Flutter complets
- Build APK/IPA
- Firebase Deploy
- Google Play upload
- App Distribution

## Pipeline Intégré

### Orchestration
- Détection changements automatique
- Tests E2E coordination
- Audit sécurité OWASP  
- Performance Lighthouse
- Déploiements coordonnés

## Release Management

### Automatisation
- Releases GitHub automatiques
- Artifacts et Docker tags
- Deploy production avec validation
- Rollback automatique si échec

## Monitoring

### Post-deployment
- Health checks automatiques
- Performance monitoring
- Error tracking integration
- User metrics dashboard
