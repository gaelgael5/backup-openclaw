# Analyse Concurrentielle MyCoach — 2 Mars 2026

## Synthèse exécutive

- **Trainerize domine** avec 16k+ coaches actifs, pricing tiered €0-200/mois, forte présence internationale
- **Marché B2B fragmenté** : solutions premium (TrueCoach) vs freemium (Trainerize) vs génériques (Calendly+Stripe)
- **Opportunité France** : concurrents focalisés sur marchés anglo-saxons, peu d'adaptation locale
- **Gap pricing** : absence d'offre €20-50/mois pour coachs indépendants français
- **UX fragmentée** : solutions génériques nécessitent 3-4 outils séparés

## Concurrents analysés

### Trainerize (Concurrent direct principal)
- **Positionnement** : Plateforme tout-en-un pour coaches en ligne et studios
- **Prix** : Gratuit (1 client) → $9/mois (2 clients) → $19.80/mois (jusqu'à 200 clients) → $225/mois (Studio, 500 clients)
- **Forces** : Écosystème complet (workout builder AI, nutrition, payments Stripe), 16k+ coaches actifs, apps mobile iOS/Android
- **Faiblesses** : Prix en USD, interface complexe pour débutants, support client limité sur plans bas
- **Avis utilisateurs** : Pas d'avis consultables directement, mais forte adoption (16k+ coaches, 250k+ clients)
- **Source** : https://www.trainerize.com/pricing/ — consulté le 02/03/26

### TrueCoach (Concurrent direct premium)
- **Positionnement** : Software premium pour personal trainers professionnels
- **Prix** : Non public, essai gratuit 14 jours, probablement $50-150/mois selon taille
- **Forces** : Focus qualité UX, intégration MyFitnessPal, messaging avancé, 20k+ trainers actifs
- **Faiblesses** : Prix cachés (signe de tarification élevée), pas de pricing transparent, acquisition moins accessible
- **Avis utilisateurs** : 20k+ trainers, témoignages positifs sur UX et simplicité
- **Source** : https://truecoach.co/pricing/ — consulté le 02/03/26

### MyFitnessPal Premium (Concurrent indirect)
- **Positionnement** : App nutrition tracking avec fonctionnalités coaching limitées
- **Prix** : €79.99/an (Premium) ou €99.99/an (Premium+)
- **Forces** : Énorme base utilisateurs, tracking nutrition avancé, meal planner, prix B2C accessible
- **Faiblesses** : Pas de gestion coach-client native, focus nutrition uniquement, pas de booking/paiements
- **Avis utilisateurs** : App grand public très utilisée, mais feedback coaching pro limité
- **Source** : https://www.myfitnesspal.com/premium — consulté le 02/03/26

### SportEasy (Concurrent indirect français)
- **Positionnement** : Gestion d'équipes et clubs de sport amateur
- **Prix** : Non affiché, probablement freemium
- **Forces** : 20k clubs actifs, 4.5M utilisateurs, adaptation France/amateurs, recognition locale
- **Faiblesses** : Focus équipes/clubs vs coaching individuel, pas de paiements intégrés
- **Avis utilisateurs** : Forte adoption amateur (4.5M users), témoignages positifs dirigeants clubs
- **Source** : https://www.sporteasy.net/fr/ — consulté le 02/03/26

### Calendly + Stripe (Solution générique)
- **Positionnement** : Booking + paiements séparés, assemblage DIY
- **Prix** : Calendly €0-12/mois + Stripe 1.4% + 0.25€ par transaction
- **Forces** : Prix bas, flexibilité totale, outils matures et fiables
- **Faiblesses** : UX fragmentée (3-4 outils), pas de gestion client native, configuration technique complexe
- **Avis utilisateurs** : Calendly très bien noté pour simplicité booking, Stripe référence paiements
- **Source** : https://calendly.com/pricing — consulté le 02/03/26

## Opportunités de différenciation

### 1. **Positionnement géographique France-first**
- **Insight** : Concurrents principaux (Trainerize, TrueCoach) optimisés pour marchés anglo-saxons
- **Opportunité** : Interface française native, adaptation réglementaire RGPD, support timezone Europe
- **Avantage concurrentiel** : SportEasy prouve la viabilité d'une solution française pour le sport

### 2. **Gap pricing €20-50/mois pour indépendants**
- **Insight** : Trainerize saute de $9 (2 clients) à $19.80 (200 clients), TrueCoach cache ses prix premium
- **Opportunité** : Offre intermédiaire €25-35/mois pour 10-30 clients (sweet spot indépendants)
- **Avantage concurrentiel** : Pricing transparent et adapté au marché français

### 3. **UX simplifiée pour coachs débutants**
- **Insight** : Trainerize complexe avec AI/automations avancées, TrueCoach premium/professionnel
- **Opportunité** : Onboarding simplifié, workflow coach débutant → expérimenté progressif
- **Avantage concurrentiel** : Réduction friction adoption, focus facilité d'usage

### 4. **Intégration paiements française**
- **Insight** : Concurrents utilisent Stripe (US-centric), solutions génériques fragmentées
- **Opportunité** : Paiements natifs CB/SEPA, facturation française automatique
- **Avantage concurrentiel** : Compliance fiscale automatique pour indépendants français

### 5. **Spécialisation coaching sportif vs généraliste**
- **Insight** : Trainerize généraliste (fitness/nutrition), SportEasy amateur/collectif
- **Opportunité** : Workflows spécifiques coaching sportif individuel (séances, progressions, objectifs)
- **Avantage concurrentiel** : Features métier vs plateforme générique

## Menaces

### 1. **Domination Trainerize sur segment international**
- **Risque** : Trainerize pourrait localiser interface française et pricing EUR
- **Mitigation** : Lancer rapidement avec avantage first-mover France

### 2. **Entrée acteurs français établis**
- **Risque** : SportEasy ou nouveau entrant français sur coaching individuel
- **Mitigation** : Focus différenciation tech et UX vs simple localisation

### 3. **Race to bottom pricing**
- **Risque** : Guerre des prix si concurrents baissent tarifs pour conquête France
- **Mitigation** : Value proposition claire au-delà du prix (UX, compliance, support)
