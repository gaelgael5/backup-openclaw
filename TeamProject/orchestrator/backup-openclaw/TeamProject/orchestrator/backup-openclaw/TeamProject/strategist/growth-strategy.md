# Stratégie de Croissance MyCoach 0-12 Mois — 2 Mars 2026

## Synthèse exécutive

- **Objectif Y1** : 500-700 coaches actifs, €200k-250k ARR, product-market fit validé
- **3 phases** : Launch niche (M1-M4) → Scale acquisition (M5-M8) → Optimize retention (M9-M12)
- **Channel focus** : Community-driven growth (70%) + Content marketing (20%) + Partnerships (10%)
- **Investment** : €150k marketing Y1 (€300-400 CAC, LTV/CAC 5:1, payback 8 mois)
- **Success metric** : 25% conversion gratuit→payant, <8% churn mensuel, NPS >50

## Phase 1 : Launch Niche (M1-M4)

### Objectifs phase 1
- **Clients** : 150-200 coaches payants
- **Revenue** : €40k-60k ARR
- **Focus** : Product-market fit coaching féminin + reconversion pro
- **Investment** : €30k marketing (€200 CAC target)

### Go-to-market channels

#### **Community Building (60% effort)**
- **Facebook Groups coaching féminin** : 15-20 groupes actifs 500+ membres
  - Content : Templates séances post-partum, gestion client enceinte
  - Engagement : 3-5 posts/semaine, réponses questions, partage expertise
  - Conversion : Invitation essai gratuit via DM, pas de spam public
  - ROI : 80-120 signups/mois, 25% conversion = 20-30 payants/mois

- **LinkedIn coaching professionnel** : Network reconversion + certification
  - Content : Articles business coaching, témoignages success stories
  - Outreach : Connection coachs récemment certifiés, proposition aide setup
  - Conversion : Consultation gratuite 30min → démonstration outil
  - ROI : 40-60 consultations/mois, 40% conversion = 15-25 payants/mois

#### **Content Marketing (25% effort)**
- **Blog SEO coaching business** : 2 articles/semaine
  - Keywords : "logiciel coach sportif", "gestion client coaching", "facturation coach"
  - Content : Guides pratiques, comparatifs outils, templates business
  - Distribution : Newsletter, social media, forums coaching
  - ROI : 500-800 visits/mois M4, 3% conversion = 15-25 signups/mois

- **YouTube tutorials** : 1 video/semaine
  - Content : Screencast features MyCoach, interviews coachs utilisateurs
  - SEO : "comment gérer clients coaching", "logiciel coaching français"
  - Engagement : Commentaires, liens description vers essai gratuit
  - ROI : 200-400 views/video M4, 2% conversion = 8-15 signups/mois

#### **Partnerships (15% effort)**
- **Écoles formation coaching** : 5 partnerships établies
  - Value : Module "gestion business" avec MyCoach dans cursus
  - Benefits : 3 mois gratuits nouveaux certifiés, commission école 10%
  - Target : IFOCOP, CNFDI, formations coaching sportif certifiantes
  - ROI : 50-100 nouveaux coaches/mois via écoles, 15% conversion

### Metrics phase 1
- **Signups total** : 800-1200 (mix channels)
- **Conversion payant** : 25% = 200-300 coaches payants
- **CAC blended** : €200 (community €150, content €300, partnerships €100)
- **Churn mensuel** : 12% (acceptable phase early adoption)

## Phase 2 : Scale Acquisition (M5-M8)

### Objectifs phase 2
- **Clients** : 350-500 coaches payants (cumul)
- **Revenue** : €120k-180k ARR  
- **Focus** : Scale channels validés + expansion segments
- **Investment** : €60k marketing (€250 CAC target avec scale)

### Channel optimization et expansion

#### **Community Marketing Scale (50% effort)**
- **Automation tools** : Buffer/Hootsuite pour posting multi-groupes
- **User-generated content** : Incentives coaches partager success stories
- **Influencer partnerships** : Top 10 coaches French Instagram/TikTok collaboration
- **Events** : Webinars mensuels "Business Coaching", co-organization avec influencers
- **ROI** : 300-500 signups/mois, 25-30% conversion = 75-150 payants/mois

#### **Content Marketing Scale (30% effort)**
- **SEO acceleration** : 15-20 articles/mois, guest posting blogs coaching
- **Lead magnets** : Templates gratuits (contrats, planning) contre email
- **Email marketing** : Newsletter hebdo, sequences onboarding automatisées
- **Podcast appearances** : 5-10 podcasts coaching/entrepreneuriat français
- **ROI** : 800-1200 visits/mois, 4-5% conversion = 30-60 signups/mois

#### **Paid Acquisition Launch (15% effort)**
- **Google Ads** : Keywords "logiciel coach", "gestion client coaching"
- **Facebook/Instagram Ads** : Retargeting visiteurs site, lookalike users payants
- **LinkedIn Ads** : Targeting coachs certifiés, entrepreneurs fitness
- **Budget** : €1500-2500/mois, CPC €2-5, conversion 3-5%
- **ROI** : 200-400 signups/mois, 20% conversion = 40-80 payants/mois

#### **Referral Program Launch (5% effort)**
- **Mechanics** : Coach parraine → 1 mois gratuit chacun après 2 mois payant
- **Implementation** : Dashboard referral, tracking attribution, rewards automatiques
- **Promotion** : Email existing users, in-app notifications, social sharing
- **ROI** : 15-25% users refer, 10-20 nouveaux payants/mois via referral

### Metrics phase 2
- **Signups total** : 1200-1800/mois (scale channels)
- **Conversion payant** : 23-25% (slight decrease due paid acquisition)
- **CAC blended** : €250 (community €180, content €250, paid €400, referral €50)
- **Churn mensuel** : 10% (optimization retention parallel)

## Phase 3 : Optimize Retention (M9-M12)

### Objectifs phase 3
- **Clients** : 500-700 coaches payants (cumul)
- **Revenue** : €200k-250k ARR
- **Focus** : Retention optimization, product-led growth, upselling
- **Investment** : €60k marketing (maintain CAC, improve LTV)

### Retention et expansion revenue

#### **Onboarding Optimization (30% effort)**
- **Success path definition** : 3 clients créés + 5 séances bookées = success
- **Guided onboarding** : Checklist in-app, progress tracking, celebrations
- **Personal touch** : Onboarding calls nouveaux payants, success manager assign
- **Early warning** : Alerts usage baisse, outreach proactif avant churn
- **Impact** : Churn M1 reduction 15% → 10%, time-to-value -50%

#### **Feature Adoption Drive (25% effort)**
- **In-app education** : Tooltips, feature discovery, usage analytics
- **Webinars features** : Monthly demos nouvelles features, Q&A
- **Success stories** : Case studies clients ROI, testimonials video
- **Upselling automation** : Plan upgrade suggestions basées usage patterns
- **Impact** : Plan Expert adoption +40%, feature usage +60%

#### **Community Engagement (25% effort)**
- **User forum launch** : Private Facebook group users MyCoach
- **Coaching business content** : Weekly tips marketing, pricing, client retention
- **Peer learning** : Connections between similar coaches, mastermind groups
- **Recognition program** : Coach du mois, success spotlights
- **Impact** : Engagement +80%, churn reduction community users -30%

#### **Product-Led Growth (20% effort)**
- **Viral mechanics** : Client booking pages avec branding MyCoach
- **Client experience** : Booking confirmation emails avec coach recommendation
- **Coach tools sharing** : Templates séances, programmes partageables
- **API ecosystem** : Integrations comptabilité, newsletters, CRM
- **Impact** : 10-15% signups via viral, product stickiness +40%

### Metrics phase 3
- **New signups** : 800-1200/mois (focus retention vs pure acquisition)
- **Conversion payant** : 27-30% (onboarding improvement)
- **Churn mensuel** : 6-8% (industry best-in-class)
- **Upselling rate** : 20-25% Coach→Expert, 15% Expert→Studio

### Risk mitigation

#### **Competition response**
- **Risk** : Trainerize localisation française
- **Mitigation** : Community moat, faster feature development, premium positioning

#### **Channel dependency**  
- **Risk** : Facebook algorithm changes
- **Mitigation** : Channel diversification, owned media (email, blog), direct traffic

#### **Churn acceleration**
- **Risk** : Product-market fit issues
- **Mitigation** : Monthly cohort analysis, user feedback loops, feature prioritization

#### **CAC inflation**
- **Risk** : Paid channels saturation
- **Mitigation** : Organic channel focus, referral optimization, LTV improvement
