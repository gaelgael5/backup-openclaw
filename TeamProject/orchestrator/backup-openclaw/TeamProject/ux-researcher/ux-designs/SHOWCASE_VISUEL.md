## ⚡ EFFETS VISUELS SIGNATURES

### 🌟 Glassmorphism Hierarchy
```
NIVEAU 1 - Background Glass
├─ Blur: 40px | Opacity: 5% | Usage: Page backgrounds
├─ Coach: Deep space + orange mesh
└─ Client: Light blue + cyan waves

NIVEAU 2 - Card Glass  
├─ Blur: 25px | Opacity: 8% (coach) / 50% (client)
├─ Border: Themed @20% | Shadow: Contextual
└─ Usage: Main content containers

NIVEAU 3 - Component Glass
├─ Blur: 15px | Opacity: 12% | Border: @15%
├─ Usage: Buttons, inputs, navigation
└─ Interactive states: Glow intensify

NIVEAU 4 - Accent Glass
├─ Blur: 10px | Opacity: 15% | Border: @25%  
├─ Usage: Active states, highlights
└─ Special: Pulsing + morphing effects
```

### 🔥 Glow & Neon System
```
COACH THEME GLOWS:
├─ Orange Primary: #FF6B35AA - Command/control elements
├─ Purple Accent: #8B5CF6AA - Analytics/data elements  
├─ Cyan Tech: #06FFA5AA - System status/connectivity
└─ Red Alert: #FF4444AA - Urgent notifications/errors

CLIENT THEME GLOWS:
├─ Cyan Primary: #00D2FFAA - Progress/achievements
├─ Pink Energy: #FF4081AA - Social/motivation elements
├─ Purple Magic: #7B68EEAA - Gamification/rewards  
└─ Green Success: #00FF94AA - Completed goals/wins

GLOW INTENSITIES:
├─ Idle: 20px spread, 30% opacity

└─ Special: Multi-layer + pulsing animation
```
