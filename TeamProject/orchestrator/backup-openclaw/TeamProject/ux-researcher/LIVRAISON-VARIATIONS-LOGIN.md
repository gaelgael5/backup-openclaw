## 🚀 COMMENT UTILISER

### Générer les PNG

1. **Installer les dépendances :**
   ```bash
   npm install puppeteer
   ```

2. **Exécuter la conversion :**
   ```bash
   node convert-to-png.js
   ```

3. **Récupérer les PNG :**
   ```
   png-outputs/
   ├── login-coach-orange-v1.png
   ├── login-coach-v1-variation-a.png
   ├── login-coach-v1-variation-b.png
   ├── login-coach-v1-variation-c.png
   ├── login-coach-v1-variation-d.png
   └── login-coach-v1-variation-e.png
   ```
