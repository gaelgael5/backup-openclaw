### A/B tests prioritaires

2. **Screenshot order** : Dashboard first vs Planning first vs Social proof first

### Monitoring hebdomadaire
- **Search rankings** : Position mots-clés target
- **Conversion rate** : Install → trial → payant
- **Reviews sentiment** : Analyse feedback utilisateurs  
- **Competitor tracking** : New apps, features, pricing

### Signaux d'alerte
- **Download decline** >20% WoW sans raison externe
- **Rating drop** <4.0 nécessite action immédiate  
- **Keyword ranking** sortie top 50 sur mots-clés primaires
- **Review velocity** <2 reviews/semaine en phase growth
