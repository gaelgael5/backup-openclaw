# 📋 PROCÉDURE DE DÉLÉGATION - Estimation délai obligatoire

## Directive User (3 mars 2026, 16:45)

**"Quand je te confie une tâche. Demande toujours à l'agent à qui tu la confies de commencer par une estimation du délai pour obtenir le livrable."**

## Nouvelle Procédure OBLIGATOIRE

### 1. ESTIMATION DÉLAI EN PREMIER
Quand je délègue une tâche à un agent, je dois TOUJOURS inclure dans le brief :

```
AVANT TOUTE CHOSE : Estime d'abord le délai nécessaire pour ce livrable.
Fournis une estimation précise (min-max) avant de commencer le travail.
```

### 2. ATTENDRE L'ESTIMATION
- Agent doit répondre avec estimation AVANT de commencer
- Ne pas lancer le travail sans estimation validée
- Communiquer l'ETA à l'utilisateur immédiatement

### 3. VALIDATION DÉLAI
- Présenter l'estimation à l'utilisateur
- Attendre validation ou ajustement du scope
- Lancer le travail seulement après accord

### 4. SUIVI
- Surveiller le respect des délais estimés
- Alerter si dépassement prévu
- Améliorer les estimations avec l'expérience

## Application Immédiate

Cette procédure s'applique à TOUTES les futures délégations d'agents.
